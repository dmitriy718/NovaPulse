# Glossary

- Confluence: multiple strategies agreeing on direction/setup.
- Confidence: a 0..1 score used to decide if a signal is tradable.
- OBI: order book imbalance signal.
- Book score: combined microstructure score derived from order book analysis.
- Paper mode: simulated trading.
- Live mode: real orders on an exchange.
- Stale data: market data not updated recently; trading should avoid stale pairs.
